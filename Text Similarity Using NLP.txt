🚀 Overview

This project implements a comprehensive text similarity system using multiple Natural Language Processing (NLP) techniques. It compares traditional, knowledge-based, and modern embedding-based approaches to evaluate how machines understand textual similarity.

The project demonstrates how different algorithms perform across word-level and sentence-level similarity tasks, highlighting strengths and limitations of each method.

✨ Features

Edit Distance

Word2Vec (via spaCy)

🧪 Methodology

The project combines three generations of NLP approaches:

Rule-Based Approach
Edit Distance (string similarity)
Knowledge-Based Approach
WordNet (semantic relationships)
Embedding-Based Approach
Word2Vec via spaCy (contextual similarity)

📊 Key Insights

Traditional methods (Edit Distance, Jaccard) fail to capture meaning
WordNet improves semantic understanding but has limitations
Embedding-based models (Word2Vec) provide better contextual similarity
Demonstrates the lexical vs semantic gap in NLP systems
